# 🧠 Flashcard Quiz App

A Flutter-based Flashcard Quiz App for learning using interactive flashcards.

---

## 🚀 Features

- 📚 Question & Answer flashcards
- 🔄 Tap to flip card
- 🎯 Simple UI design
- ➕ Easy to extend questions

---

## 🛠️ Tech Stack

- Flutter
- Dart

---

## 📂 Project Structure

lib/
 ├── models/
 ├── data/
 ├── screens/
 ├── widgets/
 └── main.dart

---

## ▶️ How to Run

flutter pub get  
flutter run
